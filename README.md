# Inteligenci·AR

Landing de validación B2B — "soft landing" para empresas extranjeras que aterrizan,
operan y exportan desde Argentina, con foco en registrar la futura **AI-entity**.
Next.js 16 (App Router) + Tailwind v4. Sin backend.

> Antes era **Landar**. Renombrado a **Inteligenci·AR** (dominio `inteligenci.ar`).
> El remoto git todavía se llama `Landar-landing`.

## Desarrollo

```bash
npm run dev      # http://localhost:3000
npm run build    # build de producción (verificar antes de deploy)
npm run start    # servir el build localmente
```

Contexto completo y convenciones: ver `../CLAUDE.md` (raíz del workspace).
